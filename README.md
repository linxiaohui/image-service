# image-service
Dockers for Image Service

# 动机
在进行与图片处理有关内容的学习和调研过程中，发现一些使用上的问题：
   1. 相关功能依赖库的版本等环境问题
   2. 预训练模型下载难的问题
   3. 接口使用方式的问题

# 目标
为解决上述问题，方便相关功能的使用，建立本项目：
   1. 收集OpenCV、dlib、深度学习 等技术进行图片处理的相关技术和方法
   2. 使用docker的方式组织
   3. 梳理预训练模型的下载方式以便于获取
